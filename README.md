# AI Communication Coach - Nirmaan Intern Case Study

## Project Overview
This tool is an AI-powered scorer that evaluates self-introduction transcripts based on the Nirmaan AI Rubric. It analyzes 5 key areas: Content, Speech Rate, Grammar, Clarity, and Engagement.
